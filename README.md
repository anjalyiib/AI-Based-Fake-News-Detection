## AI-Based-Fake-News-Detection

PROJECT OVERVIEW

This is a streamlit-based AI news analysis application that uses the meta LLaMa-3-8B-Instruct model via the Hugging Face inference API.
The system allows users to:

1.Detect fake or real news
2.Summarise news articles
3.Extract key points
4.Ask news-related questions in a chatbot format.

FEATURES

1.News Chatbot Tab

-Answer news-related questions
-Identify political bias(Left/Right/Center)

2.Fake News Detection Tab

-Checks news authenticity
-Provides detailed explanation

3.Summary and key points tab

-Summarise news in 4-5 lines
-Lists key points in bullets.

TECHNOLOGIES USED

-Python
-Streamlit
-Hugging Face Hub
-Meta LLaMA-3-8B-Instruct
